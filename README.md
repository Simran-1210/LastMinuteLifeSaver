# Last-Minute Life Saver

An AI-powered productivity companion built with Gemini AI that helps 
you never miss a deadline.

## Features
- Gemini AI task prioritization
- Gmail deadline extraction  
- Google Calendar auto-scheduling
- Firebase real-time sync
- Google OAuth authentication

## Tech Stack
- Next.js 14, TypeScript, Tailwind CSS
- Gemini 2.0 Flash (Google AI)
- Firebase Auth + Firestore
- Google Calendar API + Gmail API
- Deployed on Google Cloud Run
